순서
1. 게임 시작
  - bGameFlag = true;
  - Timer = enable;
  - 시작 버튼 비활성화
  - 입력 버튼 활성화
  - 결과 확인 비활성화
  - 포기 버튼 활성화
  - 숫자 입력 버튼 활성화

2. 문제 생성
  - 랜덤 숫자, 심볼 생성 후 표시

3. 사용자 입력 값 가져오기

4. 정답확인

5. 게임 끝
  - bGameFlag = false;
  - Timer = disable;
  - 시작 버튼 활성화
  - 입력 버튼 비활성화
  - 결과 확인 비활성화
  - 포기 버튼 비활성화
  - 숫자 입력 버튼 비활성화
  - 게임 결과 표시
